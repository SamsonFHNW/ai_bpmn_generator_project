# BPMN Engine

This folder contains modular utilities for BPMN 2.0 process validation, ML analysis, and export.

- validation_utils.py: BPMN structure validation
- ml_analysis.py: ML-based process health analysis
- export_utils.py: BPMN JSON to XML export
- bpmn_schema.json: BPMN 2.0 JSON schema
- process_examples.json: Example BPMN processes
- **init**.py: Package initializer
